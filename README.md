# 🌿 Meraki — Dil ki Baat, Bina Filter

**Meraki** is a mental wellness and journaling web app with a nature-inspired design. It encourages open self-expression with a calming, leaf-themed UI.

---

## ✨ Features

- 🍃 **Floating Leaf Animations** — Gentle falling leaf particles across the page
- 🌙 **Dark Mode** — Built-in light/dark theme toggle
- 💚 **Nature-Inspired Palette** — Calming green and cream color scheme
- 🫁 **Breathing Circle** — Guided breathing animation for relaxation
- 📱 **Fully Responsive** — Mobile-first design with Inter & Poppins typography
- ⚡ **Single-File App** — Self-contained HTML with Tailwind CSS (CDN)

---

## 🚀 Getting Started

```bash
git clone https://github.com/rishitmanktala/merakiweb.git
# Open index.html in a browser
```

| Layer | Technology |
